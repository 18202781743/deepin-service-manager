# dde-service-manager
DDE 服务管理器

## Dependencies
### Build dependencies

- libgtest-dev

- libkf5wayland-dev

### Runtime dependencies

- libkf5wayland

## Installation

### Build
$ mkdir build
$ cd build
$ cmake ..
$ make

### Install

$ sudo make install
